Active Server Pages

Active Server Pages is a web application framework or web development platform to make a dynamic websites it was released in 2002. Written in C#, Visual Basic.net, Jscript, or j# A powerful tool for making dynamic and interactive Web pages. An ASP file can contain text, HTML tags and scripts. Scripts in an ASP file are executed on the server. Used by IIS (Internet Information Services)
